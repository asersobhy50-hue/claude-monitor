# Claude Monitor 📊

تطبيق سطح مكتب (Desktop Widget) مبني بتقنية Electron لمراقبة استهلاك حساب **Claude Pro** الخاص بك بشكل حي ومباشر. يسحب التطبيق البيانات برمجياً وبشكل آمن من حسابك ليظهرها في واجهة أنيقة (Dark Glassmorphism).

## ✨ المميزات
- قراءة حية لنسبة استهلاك (Current Session, All Models, Claude Design).
- معرفة موعد تجديد الباقة (Reset Time).
- تحديث تلقائي يعمل في الخلفية كل 3 دقائق.
- واجهة صغيرة (Widget Mode) تلتصق بالشاشة لمراقبة الاستهلاك باستمرار.
- آمن 100%: يعمل على جهازك فقط ولا يرسل بياناتك لأي سيرفر خارجي.

## 🚀 كيفية التشغيل لمن يحمل الكود

إذا قمت بتحميل الكود من GitHub، اتبع الخطوات البسيطة التالية لتشغيله على جهازك:

### 1. المتطلبات الأساسية
يجب أن يكون برنامج **[Node.js](https://nodejs.org/)** مثبتاً على جهازك.

### 2. تثبيت المكتبات (Install Dependencies)
افتح موجه الأوامر (Terminal أو CMD) داخل مجلد المشروع واكتب الأمر التالي لتحميل المكتبات اللازمة لتشغيل التطبيق:
```bash
npm install
```

### 3. تشغيل التطبيق (Run the App)
بعد انتهاء التثبيت، اكتب هذا الأمر لتشغيل البرنامج:
```bash
npm start
```

## 📦 كيفية تحويله إلى برنامج تنفيذي (.exe)
إذا أردت استخراج التطبيق كبرنامج قابل للتثبيت على أي جهاز ويندوز لا يمتلك Node.js، فقط اكتب:
```bash
npm run build
```
سيقوم هذا الأمر ببناء التطبيق وإنشاء مجلد اسمه `dist` ستجد بداخله ملف الـ `Setup.exe` الخاص بالتطبيق!

## ⚠️ ملاحظة هامة
التطبيق يعتمد في عمله على الهيكل الحالي لموقع `claude.ai`. إذا قامت شركة Anthropic بتغيير شكل موقعها الجذري، قد تحتاج إلى تحديث كود الـ (Scraping) الموجود في ملف `main.js`.
